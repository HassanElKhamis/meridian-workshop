**Prepared for:** Meridian Components, Inc.
**RFP:** MC-2026-0417 — Inventory Dashboard Modernization

---

## Pricing Model

We offer **fixed-fee per phase** with a **not-to-exceed cap**. You know the maximum investment for each phase before it begins. If scope grows within a phase, we absorb it up to the cap. Changes that exceed the cap are proposed as a separate change order — you approve before any additional cost accrues.

## Assumptions

- Pricing assumes the defect count in Reports is approximately 8 (as stated in the RFP). Up to 12 defects are absorbed within Phase 1's not-to-exceed cap. Beyond that, critical items are handled in a complimentary follow-up sprint.
- Pricing assumes Meridian provides access to a staging environment for test validation. If not available, we will set one up at an additional cost of $2,000.
- Travel is not included. All communication is remote (video call, Slack/Teams).
- Pricing is valid for 60 days from proposal submission.
