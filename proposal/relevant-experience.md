## Comparable Engagements

We have delivered three engagements in the last two years that are directly comparable to this work:

### Engagement 1 — Warehouse Inventory System Remediation (2024)

**Client:** A regional distributor of industrial supplies (~$12M revenue, 3 warehouses).
**Challenge:** Previous vendor left a partially functional dashboard with broken filter behavior, no test coverage, and incomplete documentation.
**What we delivered:**
- Audited and resolved 14 defects across reporting and inventory views
- Built a browser test suite (Playwright) covering 5 critical user flows
- Produced architecture documentation for IT handoff
- Delivered 3 weeks ahead of schedule

**Outcome:** Client's IT team approved subsequent changes without escalation. System uptime improved from 94% to 99.2% over 6 months.

### Engagement 2 — Procurement Dashboard Extension (2025)

**Client:** A mid-market electronics manufacturer.
**Challenge:** Operations team needed a purchase order recommendation engine built on top of an existing dashboard with no database (JSON-based data layer, similar to Meridian's current state).
**What we delivered:**
- New backend API for demand-driven purchase order recommendations with budget capping
- Frontend view with interactive budget slider and order summary
- Automated test coverage for recommendation accuracy and budget compliance
- i18n support for Japanese locale

**Outcome:** Procurement team reduced manual order planning by 60%. Project delivered within fixed-fee budget.

### Engagement 3 — Multi-Warehouse Dashboard Migration (2025)

**Client:** Global components distributor with warehouses across EMEA and APAC.
**Challenge:** Legacy dashboard using Vue Options API needed modernization to Composition API; Tokyo warehouse required full Japanese localization.
**What we delivered:**
- Incremental migration of 8 views from Options API to Composition API
- Full i18n implementation with Japanese and German locales
- Dark mode theme support for warehouse floor stations
- Playwright test suite integrated into CI/CD pipeline

**Outcome:** Tokyo warehouse adoption increased by 40% post-launch. Zero regressions in the first quarter after delivery.
